A modern Doctor Appointment Web Application built using React.js that allows users to search and filter doctors dynamically based on multiple criteria like availability, date, time, location, and disease.
🚀 Features
🔍 Smart Doctor Search
Search doctors in real-time
Dynamic results update instantly without page reload
Search based on:
Doctor name
Disease / specialization
Location
⚙️ Advanced Filtering
Filter doctors based on:
✅ Availability (Available / Not Available)
📅 Date selection
⏰ Time slots
📍 Location
🩺 Disease / specialization
Multiple filters can be applied together
👨‍⚕️ Dynamic Doctor Listings
Displays doctor cards with:
Name
Specialization
Availability status
Location
Timing
Data rendered dynamically using React state
⚡ Fast & Responsive UI
Built with React for smooth user experience
Instant filtering without reloading
Fully responsive design for mobile and desktop
